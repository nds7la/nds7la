# 🚀 GOOD NEWS - Гайд для обычного человека

## Шаг 1: Скачай два приложения (как обычные программы)

### 1.1 Node.js (это нужно для работы проекта)
1. Зайди сюда: https://nodejs.org
2. Кликни большую зелёную кнопку "LTS" (это стабильная версия)
3. Скачается файл `node-v20.xxx.msi` (для Windows) или `.dmg` (для Mac)
4. Открой файл и установи как обычную программу (Next, Next, Finish)
5. **Перезагрузи компьютер**

Проверка работает ли:
- Открой "Командную строку" (Windows) или "Терминал" (Mac)
- Напиши: `node --version`
- Должно показать версию (например `v20.10.0`)

### 1.2 PostgreSQL (это база данных)
1. Зайди сюда: https://www.postgresql.org/download
2. Выбери свою ОС (Windows/Mac)
3. Скачай установщик
4. Установи как обычную программу
5. **Запомни пароль** который ты придумаешь при установке (напиши его на бумаге!)

Проверка:
- Откроется приложение "pgAdmin" (такой веб-интерфейс для базы)
- Если открылось - всё отлично

## Шаг 3: Открой "Командную строку" (самое страшное, но не страшно!)

**Для Windows:**
1. Нажми кнопку Windows
2. Напиши "cmd" 
3. Нажми Enter
4. Откроется чёрное окно с белым текстом (это нормально!)

**Для Mac:**
1. Найди приложение "Терминал"
2. Открой его

Теперь в этом окне напиши:
```
cd C:\Good-News
```
(если у тебя другая папка - замени путь)

Нажми Enter.

---

## Шаг 4: Установи зависимости (автоматически)

В том же чёрном окне напиши:
```
npm install
```

Нажми Enter и жди... Это займет 2-5 минут. Будет много текста - это нормально!

Когда закончится - в папке появится папка `node_modules` (большая, не трогай её)

---

## Шаг 5: Настрой подключение к базе данных

1. В папке `C:\Good-News` найди файл `.env.example`
2. Скопируй его (правый клик → Копировать)
3. Вставь в ту же папку (правый клик → Вставить)
4. Переименуй с `.env.example` на `.env.local`

Открой файл `.env.local` любым текстовым редактором (даже блокнот):

Найди строку:
```
DATABASE_URL="postgresql://user:password@localhost:5432/goodnews"
```

Замени на:
```
DATABASE_URL="postgresql://postgres:ТОЙ_ПАРОЛЬ_ЧТО_ТЫ_ПРИДУМАЛ@localhost:5432/goodnews"
```

Например если пароль был "MyPassword123":
```
DATABASE_URL="postgresql://postgres:MyPassword123@localhost:5432/goodnews"
```

**Остальные строки в этом файле пока пропусти** (они станут нужны позже)

Сохрани файл (Ctrl+S)

## Шаг 6: Создай базу данных

В чёрном окне (командной строке) напиши:
```
npx prisma migrate dev --name init
```

Нажми Enter и жди. Будет спрашивать - жми "y" и Enter.

Это создаст все необходимые таблицы в базе данных.

## Шаг 9: Получи API ключи (это нужно для AI и картинок)

### 9.1 Claude API ключ (для анализа новостей)

1. Зайди сюда: https://console.anthropic.com/account/keys
2. Нажми "Create Key"
3. Скопируй ключ (он выглядит как: `sk-ant-xxxxxxxxxxxxx`)
4. Вернись в админке (http://localhost:3000/control)
5. Найди Settings → AI Settings
6. Вставь ключ в поле "ANTHROPIC_API_KEY"
7. Сохрани

### 9.2 Replicate API ключ (для генерации картинок)

1. Зайди сюда: https://replicate.com/account/api-tokens
2. Нажми "Create token"
3. Скопируй токен
4. Вернись в админке
5. Settings → Image Generation
6. Вставь токен в поле "REPLICATE_API_TOKEN"
7. Сохрани

## Шаг 10: Первая статья! 🎯

Теперь самое интересное!

### 10.1 Добавь RSS источник (откуда брать новости)

1. В админке найди меню слева "RSS Sources"
2. Нажми "+ Add Source"
3. Заполни:
   - **Name:** Например "BBC News"
   - **URL:** https://feeds.bbci.co.uk/news/rss.xml
   - **Category:** Technology
   - **Trust Score:** 95
4. Сохрани

Добавь ещё несколько источников:
- https://www.theverge.com/rss/index.xml (The Verge)
- https://feeds.arstechnica.com/arstechnica/index (Ars Technica)
- https://feeds.reuters.com/reuters/scienceNews (Reuters Science)

### 10.2 Загрузи новости

1. На главной странице (Dashboard) нажми синюю кнопку "Fetch RSS"
2. Жди 30-60 секунд
3. Должны появиться новости в разделе "Articles" → "Incoming"

### 10.3 Выбери хорошую новость

1. Иди в "Articles" → "Incoming"
2. Ищи новость с высокой оценкой позитива (зелёная полоса)
3. Кликни на неё
4. Нажми "Approve"

### 10.4 Сгенерируй контент

1. Статья перейдёт в "Approved"
2. Кликни на неё ещё раз
3. Нажми "Generate Content"
4. Жди 20-30 секунд (идёт запрос к Claude)
5. Должны появиться тексты для:
   - Telegram
   - Twitter
   - Instagram
   - Facebook
   - Pinterest
   - Website

### 10.5 Сгенерируй картинку

1. На той же странице нажми "Generate Image"
2. Жди 1-2 минуты (идёт генерация через Replicate)
3. Должна появиться красивая картинка

### 10.6 Опубликуй!

1. Перейди в "Publishing Queue"
2. Найди твою статью
3. Выбери платформу где опубликовать (пока выбери Telegram)
4. Нажми "Publish Now"
5. Статья должна появиться в Telegram канале!

## ⚠️ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

### "Не устанавливается Node.js"
- Перезагрузи компьютер
- Попробуй ещё раз
- Убедись что скачал нужную версию (32-bit или 64-bit)

### "Ошибка при npm install"
- Закрой командную строку
- Переоткрой
- Убедись что находишься в правильной папке (`cd C:\Good-News`)
- Попробуй ещё раз

### "Не работает база данных"
- Проверь что PostgreSQL установлена
- Проверь что в `.env.local` правильный пароль
- Попробуй создать базу вручную через pgAdmin

### "localhost:3000 не открывается"
- Убедись что командная строка всё ещё работает (там должен быть процесс)
- Перезагрузи
- Попробуй другой браузер

### "Не получается логиниться"
- Username: admin@goodnews.local
- Password: password123
- Проверь что пароль правильный (с маленькой "p")

## 📱 Как добавить Telegram?

### Шаг 1: Создай Telegram бота
1. Открой Telegram
2. Найди бота @BotFather
3. Напиши /newbot
4. Следуй инструкциям (выбери имя, юзернейм)
5. Получишь ключ вроде: `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`
6. **Запомни этот ключ**

### Шаг 2: Добавь в проект
1. Вернись в админку
2. Settings → Platform Connectors
3. Telegram → Edit
4. Вставь ключ в поле "TELEGRAM_BOT_TOKEN"
5. Вставь ID своего канала в "TELEGRAM_CHANNEL_ID"
6. Сохрани

### Шаг 3: Протестируй
1. Нажми "Test Connection"
2. Должно показать "Connected"
3. Готово!

Теперь при публикации сможешь выбрать Telegram.
